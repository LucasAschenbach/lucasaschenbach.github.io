Privacy Policy

Last updated: July 2, 2026

This Privacy Policy governs the use of the arXiv Classic Red Chrome Extension (the "Extension") and outlines the types of information collected by the Extension and how it is used. Please read this Privacy Policy carefully before using the Extension. By using the Extension, you agree to the terms of this Privacy Policy. If you do not agree with these terms, please do not use the Extension.

Information Collection and Use

The arXiv Classic Red Chrome Extension does not collect, store, share, sell, or transmit any personally identifiable information.

The Extension only applies local visual changes to arxiv.org pages, including restoring the classic red header color, replacing the header logo for display inside the page, and adjusting link styling. These changes happen entirely inside your browser.

The Extension does not use analytics, tracking pixels, cookies, external servers, or remote code. It does not read, save, or send browsing history, page content, account information, search queries, or any other user data.

Permissions

The Extension requests access only to arxiv.org pages so it can apply its visual styling changes on those pages. This access is used solely for the Extension's single purpose of restoring classic arXiv styling.

Data Security

The Extension is designed to prioritize user privacy and does not gather or transmit any personal data. Because no user data is collected, stored, or sent to external servers, there is no user data handled by the Extension that could be sold, shared, or misused.

Affiliation

arXiv Classic Red is an unofficial extension and is not affiliated with, endorsed by, reviewed by, or approved by arXiv, Cornell University, or Google.
